# 🍔 Flutter Food App

A simple and modern food ordering UI built using Flutter.

---

## 📱 Screens

* Home Screen
* Product Details Screen
* Favorites Screen
* Profile Screen

---

## ✨ Features

* Browse food categories
* View product details
* Add items to favorites ❤️
* Clean and modern UI

---

## 🛠 Technologies Used

* Flutter
* Dart

---

## 🚀 Run the App

flutter pub get
flutter run
